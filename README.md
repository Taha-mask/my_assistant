# مساعد صوتي شخصي لـ Windows

مشروع Python لمساعد صوتي شخصي يعمل على Windows ويدعم الأوامر الصوتية بالعربية المصرية باستخدام `SpeechRecognition`، مع رد صوتي عبر `pyttsx3`، والتحكم في بعض وظائف النظام والبرامج الشائعة.

## المميزات

- كلمة تنشيط: `يا جارفيس` أو `هاي`
- التعرف على الصوت باللغة العربية المصرية `ar-EG`
- رد صوتي محلي باستخدام `pyttsx3`
- دعم أوامر النظام: قفل الجهاز، إعادة التشغيل، وضع السكون
- فتح برامج شائعة مثل Chrome وFile Explorer وVS Code
- البحث على Google وفتح مواقع مباشرة
- يبدأ تلقائيًا مع Windows عند تفعيل auto-start
- لا يحتاج كلمة تنشيط؛ يتعامل مع الكلام الطبيعي مباشرة
- التحكم في الصوت: رفع، خفض، كتم، إلغاء الكتم
- معرفة الوقت والتاريخ
- فتح مجلدات معروفة أو مسارات مباشرة
- إغلاق برامج مفتوحة
- تسجيل لوج لكل الأوامر داخل `logs/assistant_commands.txt`
- واجهة Terminal بسيطة توضح الحالة أثناء العمل

## الملفات

- `main.py`: الكود الكامل للمساعد
- `requirements.txt`: المكتبات المطلوبة
- `logs/assistant_commands.txt`: ملف اللوج ويتولد تلقائيًا عند التشغيل

## التثبيت خطوة بخطوة

1. تأكد أن Python 3.10 أو أحدث مثبت على Windows.
2. افتح `PowerShell` داخل مجلد المشروع.
3. أنشئ بيئة افتراضية:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

4. ثبّت المكتبات:

```powershell
pip install -r requirements.txt
```

5. إذا واجهت مشكلة في تثبيت `PyAudio`:

```powershell
python -m pip install --upgrade pip wheel setuptools
pip install PyAudio
```

لو استمرت المشكلة، استخدم Wheel مناسب لإصدار Python وWindows لديك ثم ثبّته يدويًا.

## التشغيل

```powershell
python main.py
```

بعد التشغيل سيظهر لك في الـ Terminal أن المساعد جاهز، ثم قل أي سؤال أو أمر مباشرة. كلمة التنشيط اختيارية وليست مطلوبة.

## تشغيل الواجهة ولوحة التحكم

```powershell
python main.py --gui
```

الواجهة تعرض حالة المساعد، أزرار تشغيل/إيقاف الاستماع الصوتي، إرسال أوامر كتابية، مؤشرات الجهاز، حالة التشغيل التلقائي مع Windows، وآخر سجل تشغيل مباشر.

## التشغيل التلقائي مع Windows

لتفعيل التشغيل التلقائي عند تسجيل الدخول إلى Windows:

```powershell
python main.py --install-startup
```

للتحقق من الحالة:

```powershell
python main.py --startup-status
```

ولإلغاء التشغيل التلقائي:

```powershell
python main.py --remove-startup
```

## ملاحظات مهمة

- التعرف على الصوت هنا يعتمد على خدمة Google داخل مكتبة `SpeechRecognition`، لذلك يحتاج إنترنت وقت التعرف على الكلام.
- الرد الصوتي سيستخدم صوتًا عربيًا إذا كان مثبتًا في Windows. لو لم يجد صوتًا عربيًا، سيستخدم الصوت الافتراضي المتاح في النظام.
- عند بدء التشغيل التلقائي، Jarvis سيفتح مع تسجيل دخولك وسيحييك مباشرة ثم يظل مستعدًا للاستماع.
- بعض الأوامر الحساسة مثل إعادة التشغيل أو وضع السكون تطلب تأكيدًا صوتيًا قبل التنفيذ.
- يمكنك رفع الذكاء والسرعة بتغيير `OLLAMA_MODEL` في ملف `.env` إلى موديل أكبر مثل `qwen2.5:7b-instruct` أو `llama3.1:8b-instruct` إذا كان جهازك يتحمل.
- المشروع مخصص لـ Windows فقط.

## التعديل والتوسعة

- لإضافة برنامج جديد: أضفه داخل `APP_SPECS` و`APP_ALIASES` في `main.py`
- لإضافة مجلدات معروفة: أضفها داخل `FOLDER_ALIASES`
- لإضافة مواقع معروفة: أضفها داخل `SITE_ALIASES`
- لتغيير كلمة التنشيط أو سرعة الكلام: عدّل `VoiceAssistantConfig`
